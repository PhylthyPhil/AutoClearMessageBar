# Auto Clear Message Bar (QGIS Plugin)

A QGIS plugin that automatically dismisses message bar warnings and
information messages at a configurable interval.

## Features

- Automatic clearing of QGIS message bar notifications
- Configurable interval
- Enable / disable toggle
- Toolbar button
- Persistent settings

## Installation

### From QGIS Plugin Manager
1. Open QGIS
2. Go to Plugins → Manage and Install Plugins
3. Search for **Auto Clear Message Bar**
4. Install and enable

### From ZIP (manual)
1. Download the ZIP from the Releases page
2. Plugins → Manage and Install Plugins → Install from ZIP
